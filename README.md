# Service Climatização

Site institucional estático multipágina pronto para hospedagem.

## Publicação rápida

Publique todos os arquivos desta pasta na raiz da hospedagem.

- Netlify/Cloudflare Pages: usa `_headers`.
- Vercel: usa `vercel.json`.
- Apache/cPanel/Hostinger: usa `.htaccess`.

## Páginas principais

- `index.html`
- `sobre.html`
- `valores.html`
- `servicos.html`
- `pmoc.html`
- `marcas.html`
- `diferenciais.html`
- `galeria.html`
- `depoimentos.html`
- `contato.html`

Leia `DOCUMENTACAO_TECNICA.md` antes de alterar contatos, domínio, scripts de SEO ou headers de segurança.
